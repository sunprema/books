A macro

defmacro m(arg, do: blk) do
  quote do
    if unquote(arg),
      do: unquote(blk)
  end
end
# caller must: require M

Hygiene

  • Quoted vars don't leak — different context.
  • var!(x) → use the caller's x (sparingly!).
  • __CALLER__ → Macro.Env of the call site.

use = require + __using__

defmacro __using__(opts) do
  quote do
    import X
    @before_compile X
  end

Anti-patterns ✕

  • Macro where a function works.
  • Huge quote blocks → slow compiles.
  • use where import suffices.
  • Needless compile-time deps.

The rule

Write the function first. Reach for a macro only when you need an unevaluated argument, a declarative DSL, compile-time codegen, or compile-time guarantees.

🔑 Ownership rules

  • Each value has exactly one owner.
  • Owner out of scope → value dropped (auto cleanup).
  • Assign/pass a non-Copy value → move; old name dead.
  • Deep copy on demand: .clone().
  • Copy types (stack only): ints, floats, bool, char, tuples of Copy.

🔗 Borrowing

fn read(s: &String) {}      // shared
fn write(s: &mut String) {}  // exclusive

The rule: many &T XOR one &mut T — never both. No borrow may outlive its referent.

📦 Smart pointers

Box<T>heap, single owner
Rc<T>shared, 1 thread
Arc<T>shared, threads
RefCell<T>borrow check at runtime
Mutex<T>lock for shared mut

🧩 Traits

trait Area { fn area(&self)->f64; }
impl Area for Circle { … }
  • Static: fn f<T: Area>(x:&T) / &impl Area — zero-cost.
  • Dynamic: &dyn Area, Box<dyn Area> — vtable, heterogeneous.
  • #[derive(Debug, Clone, PartialEq)]

⏳ Lifetimes

fn longest<'a>(a:&'a str, b:&'a str)
    -> &'a str { … }
  • A label, not a runtime thing. Usually elided.
  • 'static = whole program (e.g. string literals).

🎯 Enums & match

match x {
    Some(v) => …,
    1 | 2 => …,        // or
    3..=9 => …,        // range
    n if n < 0 => …,   // guard
    _ => …,            // catch-all
}

Exhaustive — every case or it won't compile.

🧯 Errors

fn go() -> Result<T, E> {
    let v = may_fail()?;   // ? = unwrap or return Err
    Ok(v)
}
  • Option<T> = no null; Result<T,E> = no exceptions.
  • ? auto-converts via From; needs Result/Option return.
  • Bug → panic!/unwrap. Apps → anyhow, libs → thiserror.

🔁 Iterators

v.iter()
 .filter(|x| **x > 0)
 .map(|x| x * 2)
 .collect::<Vec<_>>();
  • iter=&T · iter_mut=&mut T · into_iter=T
  • Adapters lazy; consumers (sum,collect,fold) drive.

🧠 Closures

let add = |a, b| a + b;
move || { use_owned(data) }
Fn&capture, reusable
FnMut&mut, reusable
FnOnceby value, once

🧵 Concurrency

let c = Arc::new(Mutex::new(0));
let c2 = Arc::clone(&c);
thread::spawn(move || {
    *c2.lock().unwrap() += 1;
});
  • Send=move to thread · Sync=share by ref.
  • Channels: mpsc::channel() · tx.send/rx.recv.
  • Prevents data races, not deadlocks. Async: Tokio.

📦 Cargo

cargo new · run · build --release
cargo test · fmt · clippy · doc
cargo add <crate>
  • Cargo.toml = manifest; Cargo.lock = pinned versions.
  • Editions 2015→2024: opt-in, interoperate.

⏱ 30-second mental model

  • Ownership → who frees it.
  • Borrowing → who may read/write.
  • Lifetimes → references stay valid.
  • Traits → shared behavior, your dispatch cost.
  • Enums+match → illegal states impossible.
  • All checked at compile time, zero runtime cost.

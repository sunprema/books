Every command, crate, and gotcha from the book — distilled. Print it; tape it above your bench.

🔧 Toolchain (one-time)

$ cargo install espup espflash
$ espup install
$ source ~/export-esp.sh
# every new shell needs the source!
$ cargo install esp-generate

Xtensa (S3) needs espup. RISC-V chips (C3/C6) use plain nightly + rustup target add.

🚀 Scaffold & flash

# no_std project
$ esp-generate --chip esp32s3 app
# std project
$ cargo generate \
    esp-rs/esp-idf-template
# build + flash + monitor
$ cargo run --release
$ espflash monitor   # just watch

📨 MQTT QoS

QoSGuaranteeUse
0at most oncecheap telemetry

LWT = broker posts your "offline" msg if you drop. Retained = broker keeps last value for new subscribers.

🌳 Topic scheme

sensors/<id>/temp   up
sensors/<id>/status LWT
sensors/<id>/cmd    down
# wildcards:
sensors/+/temp  # one level
sensors/#       # all below

⚠️ Common gotchas

  • Target not found → forgot source ~/export-esp.sh.
  • Won't flash → hold BOOT, tap RST; check the port.
  • Two USB ports: native USB vs UART bridge — pick the right one.
  • Wi-Fi needs 2.4 GHz; no 5 GHz.
  • Brownout resets → power the radio from a real supply, not a weak USB hub.

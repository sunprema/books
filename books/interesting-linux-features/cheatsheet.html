Every syscall, flag, and gotcha from the book on one page — grounded in man7.org and kernel.org.

☰ fd-ification

syscallflagsdoes
eventfdEFD_SEMAPHORE8-byte counter as an fd
timerfd_createTFD_TIMER_ABSTIMEa timer as an fd
signalfdSFD_NONBLOCKsignals as readable structs
pidfd_open—a process as an fd

Gotcha: must sigprocmask(SIG_BLOCK,…) before signalfd or the signal keeps its default disposition too.

🔒 Safer opening

flagdoes
O_PATHfd with no read/write power, only *at()/stat
O_TMPFILEunnamed inode, no dir entry until linkat()
O_NOFOLLOWfail with ELOOP on a symlink, don't follow

Gotcha: O_TMPFILE needs filesystem support (ext4, btrfs, xfs, tmpfs) — not every fs allows it.

⚡ Zero-copy

syscalldoes
sendfilefile → socket, in-kernel
splicefd → fd via a pipe, ≥1 side must be a pipe
vmspliceyour memory → a pipe, no copy
teeduplicate a pipe's data, don't consume it

Gotcha: after vmsplice(), don't touch the buffer until you know the reader drained it — the pipe now points at your live memory.

💾 Anonymous memory

memfd_create(name, MFD_ALLOW_SEALING);
fcntl(fd, F_ADD_SEALS,
  F_SEAL_WRITE|F_SEAL_SHRINK|
  F_SEAL_GROW|F_SEAL_SEAL);

Gotcha: sealing is opt-in — without MFD_ALLOW_SEALING at creation, the initial seal is already F_SEAL_SEAL.

🛑 epoll modes

modefires
level-triggered (default)every call, while data remains
EPOLLETonce, only on empty→ready transition
EPOLLONESHOTonce, then disabled until EPOLL_CTL_MOD

Gotcha: EPOLLET without draining to EAGAIN strands unread data with no future notification. Ever.

🔁 io_uring

piecerole
SQEyou describe an op, write it to the SQ ring
CQEkernel writes the result to the CQ ring
IORING_SETUP_SQPOLLkernel thread polls SQ — zero syscalls

Gotcha: a large enough attack surface that some hardened environments disable it by policy — check before relying on it in a sandboxed target.

🌐 Namespaces

flagisolates
CLONE_NEWPIDthe PID tree — first process born = PID 1
CLONE_NEWNSthe mount table — own private tree via bind mounts

Gotcha: if the namespace's PID 1 dies, the kernel SIGKILLs everyone else in it — ship a real init (tini).

⚙️ Kernel tuning

knobdefaultlower it for
vm.swappiness60databases, low latency (try 1–10)
vm.vfs_cache_pressure100huge directory trees, many small files
THP modemadvise (varies)set never if you see stalls

PSI: cat /proc/pressure/{cpu,memory,io} — full > 0 means everyone is stalled, not just one task.

⏳ 30-second mental model

  • If it can happen, it can probably be an fd — and then it can sit in your epoll loop.
  • Resolve a path once (O_PATH), don't re-resolve it and trust it stayed the same thing.
  • Zero-copy means moving page references, not bytes.
  • Sealing turns "please don't" into "the kernel won't let you."
  • Containers = namespaces + cgroups, not a hypervisor.
  • Tune with evidence (PSI), not folklore (a stale blog post's magic sysctl value).

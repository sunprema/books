Performance Quick Fixes

Slow builds

  • Order Dockerfile layers: OS → deps → code
  • Copy lockfiles before source for cache hits
  • Use a pre-built base image pushed from CI
  • Avoid RUN apt-get upgrade (busts cache)

Slow file I/O (macOS)

  • Use named volumes for node_modules, target/, vendor/
  • Enable VirtioFS in Docker Desktop
  • Use tmpfs for test artifacts
  • Consider Codespaces for large repos

File watching broken

  • Set polling: "files.useExperimentalFileWatcher": true
  • Increase inotify limit: fs.inotify.max_user_watches=524288
  • Exclude heavy dirs from watchers

Debugging Checklist

Container won’t start

  1. Check build output: Dev Containers: Show Container Log
  2. List containers: docker ps -a
  3. Read logs: docker logs <id>
  4. Inspect config: docker inspect <id>
  5. Test manually: docker run -it <image> bash
  6. Try Rebuild Without Cache

The One-Page Svelte 5

Every rune, every block, every gotcha from this book — condensed. Client-side only; no SvelteKit, no SSR.

The runes

RuneDoes
$state(v)Reactive value. Deep-proxies plain objects/arrays.
$state.raw(v)No proxy — reassign whole value instead of mutating (cheaper for big data).
$state.snapshot(v)Plain-object copy, for handing to non-Svelte code.
$derived(expr)Computed value. Must be side-effect free.
$derived.by(fn)Multi-statement derivation.
$effect(fn)Side effect after DOM update; return a teardown fn.
$effect.pre(fn)Same, but runs before the DOM updates.
$effect.root(fn)Non-tracked scope, manual cleanup.
$props()Destructure a component's inputs. Defaults, renames, rest all work.
$bindable(v)Marks a prop as two-way; parent opts in with bind:.

Template blocks

#if / :else if / :elseConditional rendering.
#each list as x, i (key)Keyed iteration — always key lists that can reorder.
#await p / :then / :catchBranch on a Promise's pending/fulfilled/rejected state.
#key exprDestroy & recreate contents when expr changes.
#snippet name(args)Declare reusable, parameterized markup.
{@render name(args)}Render a snippet. Optional: {@render x?.()}
{@html str}Raw HTML injection — sanitize first, always.
{@const x = expr}Local const inside a block.

The five gotchas that bite everyone

  • Proxies stop at class instances — $state won't deep-track a class or Map/Date inside it; put $state on the class's own fields.
  • Effects that write the state they read loop forever — that's a $derived in disguise.
  • Async reads inside $effect/$derived aren't tracked — only synchronous, pre-await reads count as dependencies.
  • Module-level $state can't be exported reassignable — export an object or accessor functions, mutate don't reassign.
  • Unkeyed #each can hand a reordered list the wrong DOM node — always key lists that can change order.

30-second summary

State is a variable ($state). Computed values describe state ($derived). Everything else that isn't pure — DOM, network, third-party libs — is an effect ($effect). Props flow down ($props) and only flow up when you explicitly open that door ($bindable). Reuse markup with snippets, not slots. Share state with a module by default; reach for stores or context only when their specific contract earns it.

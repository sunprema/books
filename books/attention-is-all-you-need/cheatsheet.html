Quick ReferenceThe Transformer, One Page

The 30-second version

Throw away recurrence. Turn each token into a 512-D vector, add a positional signal, then in every layer let every word attend to every word via softmax(QKᵀ/√dk)V, run 8 such heads in parallel, follow with a per-word feed-forward net, and wrap each sub-layer in Add & LayerNorm. Stack 6 in the encoder, 6 in the (masked) decoder. Result: full parallelism + constant path length + a new translation SOTA in 12 GPU-hours.

The one equation

Attention(Q,K,V) = softmax(QK⊤ / √dk)V

1 score (dot product) · 2 scale (÷√dk, tames softmax) · 3 softmax (weights sum to 1) · 4 blend the values.

Q / K / V in one line

Query = what I'm looking for · Key = what I advertise · Value = what I hand over. All three are learned projections of the same token vector.

Each sub-layer

Residual "+x" = gradient highway · LayerNorm = stable activations · everything stays 512-wide so it stacks. FFN: max(0, xW₁+b₁)W₂+b₂, 512→2048→512.

Positional encoding

PE(pos,2i)=sin(pos/10000^{2i/d}), PE(pos,2i+1)=cos(…). Added to embeddings. Relative offset = linear function of position; extrapolates to longer sequences.

Training recipe

The legacy

Encoder-only → BERT (understand) · Decoder-only → GPT / Claude / Llama (generate — "the LLMs") · Enc–dec → T5, BART. The self-attention block is unchanged; scale + pretraining did the rest.

Chapter 05 · Domain Three

Law: A Harness With Memory

Legal work is where the memory question on the Lens dominates everything else. A matter is not a query; it's a relationship that runs for months across thousands of documents, and the value of an assistant compounds only if it remembers the matter. Stateless chatbots are amnesiacs — useless for the work that actually bills. A harness with a long-term memory extension is a different animal: it accrues context the way a good associate does.

The plays

Contract review · custom tools + RAG First-pass contract review at batch scale

A custom tool that ingests a contract, a RAG extension over your firm's clause bank and playbook, and the agent produces a redline first pass: flagged deviations from standard, missing protections, citations to your preferred language. Run it interactively for one document or in print/JSON mode across a data-room of hundreds during due diligence. The associate's hours move from reading to judging.

Risk — first pass only; a lawyer owns the advice and the sign-off.

Matter memory · long-term memory extension The matter that remembers itself

This is the flagship. A long-term memory extension gives each matter its own accruing context — parties, positions, key dates, prior drafts, what opposing counsel argued last month. Ask "where did we land on the indemnity cap?" and it answers from the matter's own history, not a generic prior. Continuity across sessions is the feature that turns a novelty into a tool a partner trusts.

Risk — matter memory is privileged material; wall it per-matter and per-client.

Precedent · RAG over precedent banks Precedent retrieval that cites your own bank

RAG over the firm's precedent and brief bank means the agent drafts from language you've already won with, not from the open internet. The retrieval tool returns the source document every time, so every suggestion is traceable to a real precedent an attorney can verify — the antidote to hallucinated citations.

Risk — verify every citation; a fabricated authority is a sanctionable event.

Print/JSON mode is the leverage

The economics of legal AI live in batch. Due diligence, document review, lease-portfolio abstraction — thousands of documents, the same question each time. Print/JSON mode makes the harness a headless reviewer you fan out across the corpus: one invocation per document, structured JSON out, ingested into your review platform. A human reviews the exceptions the agent surfaces instead of reading every page. That is the difference between a demo and a margin.

Beyond law

The memory-plus-RAG shape generalizes to every professional-services vertical with long-lived engagements: accounting and audit, management consulting, architecture and engineering, insurance claims. Wherever the work is "a case that runs for months over private documents," the same three plays apply — swap the clause bank for the audit file, the precedent bank for the claims history.

The opening

Sell continuity. Every firm has tried a generic chatbot and found it forgets everything the moment the tab closes. A harness with matter memory + a private precedent RAG + batch review is a product shaped like the actual work — and the memory of a matter is a switching cost that grows every day the client uses it.

Where you get burned

Privilege and confidentiality are absolute. Memory and RAG stores hold privileged material, so client walls, access scoping, and conflict checks are not features you add later — they are the price of entry. And unverified citations have already ended careers; the tool must make verification trivial, and the professional must still do it.

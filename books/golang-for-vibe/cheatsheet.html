Quick Reference

The Go Cheatsheet

Everything from the book on one printable page. Go 1.26 · standard library first.

🐹 Anatomy

package main
import "fmt"
func main() {
    fmt.Println("hi")
}
  • package main + func main() = runnable.
  • Capitalized = exported (public); lowercase = private.
  • Unused var/import → compile error.

📦 Variables & types

var x int = 5
y := "hi"        // infer, func-only
const Pi = 3.14

Basic: int float64 string bool byte rune. Zero values: 0, "", false, nil.

🔁 Control flow

for i := 0; i < n; i++ {}   // counted
for cond {}                // while
for i, v := range xs {}    // iterate
if x > 0 {} else {}
switch x { case 1: // no fallthrough }

for is the only loop keyword.

🗂️ Slices & maps

xs := []int{1, 2}
xs = append(xs, 3)
len(xs); xs[1:]
m := map[string]int{}
m["a"] = 1
v, ok := m["a"]   // ok=false if absent

🧩 Funcs & multi-return

func div(a, b int) (int, error) {
    if b == 0 {
        return 0, fmt.Errorf("÷0")
    }
    return a / b, nil
}

Last return value is conventionally error.

⚠️ Errors

v, err := do()
if err != nil {
    return fmt.Errorf("doing: %w", err)
}
  • %w wraps; errors.Is / errors.As inspect.
  • defer f.Close() — cleanup on any return.
  • panic = bugs only, not normal failures.

🏗️ Structs & methods

type Book struct {
    Title string `json:"title"`
}
func (b Book) Hi() string { return b.Title }
func (b *Book) Set(t string) { b.Title = t }

Pointer receiver *T to mutate.

🦆 Interfaces

type Shape interface {
    Area() float64
}

Satisfied implicitly — no implements. Keep them small. Canonical: error, io.Reader, io.Writer. Compose via embedding, not inheritance.

⚡ Concurrency

go work()              // goroutine
ch := make(chan int)
ch <- 1                // send
v := <-ch               // receive (blocks)

var wg sync.WaitGroup
wg.Add(1); go func(){ defer wg.Done() }()
wg.Wait()

Share memory by communicating. context.Context = cancel/timeout. Test with -race.

✅ The agent's graders

go build ./...compiles?
go vet ./...suspicious code
gofmt -w .one true format
go test ./...tests pass?
go test -racedata races

Feed every failure back as the next prompt. Green ≠ correct — you still review.

🧪 Tests (built in)

// in foo_test.go
func TestAdd(t *testing.T) {
    if Add(2,2) != 4 {
        t.Fatalf("want 4")
    }
}

Table-driven tests are the idiom. Run: go test ./....

🚀 Ship it

go mod init github.com/you/app
go mod tidy
go build -o app ./...
GOOS=linux GOARCH=amd64 go build ./...

One static binary, deps inside. Copy & run — nothing to install. CGO_ENABLED=0 forces pure-Go static.

🎯 Best fit / not

  • Great: CLIs, web APIs, microservices, cloud-native infra, AI plumbing.
  • Not: data science / ML training (use Python), GUIs, throwaway one-liners.

🆕 Go 1.26 (Feb 2026)

  • Green Tea GC now default (10–40% less GC overhead).
  • new(expr); self-referential generic types.
  • Faster io.ReadAll, lower cgo overhead.
  • Backward compatible — old code still builds.
🔑
The one-line philosophy

Go is small and boring on purpose. When a machine writes the code and you judge it, boring is a feature: easy to read, easy to verify, fast to compile, one binary to ship.

The 30-Second Cheatsheet

The mental model

  • Shortcut = pipeline of actions; each output feeds the next step automatically.
  • Magic Variables: every step's result is available later — tap a parameter → Select Magic Variable; drill into properties (temperature, start time…).
  • Shortcut Input = what launched it (share-sheet photo, Siri text, file). Stop and Output returns a result.
  • Compose: small shortcuts call each other via Run Shortcut.

Automation triggers

  • Time: time of day · sunrise/sunset ± offset · alarm stopped · wake-up/bedtime · Watch workout
  • Travel: arrive/leave location · CarPlay connect/disconnect
  • Communication: message or email received
  • Device: app open/close · Wi-Fi · Bluetooth · charger · battery level · Low Power · Focus on/off · NFC scan · Sound Recognition
  • Run mode: Run Immediately (silent butler) vs Run After Confirmation (anything outbound or costly)
  • Mac (Tahoe 26): folder-watch, disk, email, Focus, time triggers on the desktop too

Gotchas & safety

  • Run new shortcuts once manually to clear permission prompts before trusting them to Siri/automations.
  • Location triggers are street-sized, not room-sized — use NFC for rooms.
  • Quick Look is your debugger; silence ≠ success — end risky recipes with a ✓ notification.
  • Downloaded shortcuts: read the action list first; distrust unexplained network calls and data grabs.
  • Duplicate before major edits — no cross-session undo.

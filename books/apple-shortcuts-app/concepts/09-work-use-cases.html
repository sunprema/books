Chapter 09 · Where It Earns Its Keep, Part 2

Use Cases: Getting Work Done

Work automations save bigger chunks of time than lifestyle ones, because work is where the repetitive multi-step chores live: capturing, converting, filing, formatting, reporting. Here are the patterns that professionals actually keep using after the novelty wears off.

1 · The meeting funnel (the single best work shortcut)

Share a voice memo → filed summary
🎙
Receive audio from Share Sheet Works with Voice Memos, call recordings, WhatsApp audio.
📝
Transcribe Audio On-device speech-to-text; noticeably improved in the 26 release.
🧠
Use Model "Summarize this transcript: decisions made, action items with owners, open questions — as three lists."
📁
Create Note Title = date + first line; body = summary + full transcript; folder = "Meetings". Add reminders from the action items if you're fancy.

Why it wins: a 45-minute recording becomes a scannable record in under a minute, and the whole pipeline is private — transcription on-device, summarization on-device or Private Cloud Compute.

2 · Document capture → clean PDF

A Home Screen shortcut: Scan Document (or take photos) → Make PDF → Ask for Input for a name → Save File into the right iCloud/Files folder. Receipts, contracts, whiteboards — one tap, consistently named, already where your accountant expects them. Add Extract Text from Image and a Match Text for the total, and receipts can log themselves into a spreadsheet row via Get Contents of URL (chapter 10).

3 · Batch image work

Select any number of images in Photos or Finder, share to one shortcut: Repeat with Each → Remove Background / Resize Image / Convert Image (HEIC→JPEG/PNG) → save all to a folder. Product listings, thumbnails, slide assets — the fifty-image version takes the same one tap as the one-image version. On the Mac, expose it as a Finder Quick Action and it feels like a built-in feature.

4 · Text expander, template engine

  • Canned replies: a menu shortcut of your five most-typed responses → Copy to Clipboard. On Mac, bind it to a hotkey.
  • Daily note / stand-up template: one tap creates today's note pre-filled with date, calendar events (inserted as Magic Variables), and yesterday's unfinished items.
  • Tone polish: clipboard → Adjust Tone of Text (professional) or Proofread Text → paste-ready. The one-keystroke email cleaner.

5 · Focus, defended automatically

Trigger Work Focus turns on (itself scheduled, or fired by arriving at the office / joining office Wi-Fi) → open your task manager, start a timer, set Slack status via its Shortcuts action or API, silence the noise. When Focus turns off, a closing automation logs the session length to a note. You get automatic time tracking as a side effect.

6 · The expense one-tap

Ask for Input (amount) → Choose from Menu (category) → Text a CSV line "2026-07-06, 14.50, client lunch" → Append to File on iCloud Drive. Twelve seconds at the register beats twenty minutes of receipt archaeology at month-end. The file opens clean in Numbers or Excel.

7 · Reporting glue

Anything your team reports on a rhythm can assemble itself: a Friday 4pm automation that pulls the week's completed reminders, calendar events tagged #client, and a Screen Time summary (Get App & Website Data, new in 26), asks Use Model for a three-bullet status draft, and drops it into Mail — as a draft, because chapter 6's rule applies: anything outbound gets a human glance first.

Key Idea

The professional pattern is capture → transform → file, run from the share sheet or a hotkey. If you do a chore twice a week and it has more than four steps, it's a shortcut. The build costs fifteen minutes once; the chore cost fifteen minutes forever.

Note

Your third-party work apps — Things, OmniFocus, Drafts, Fantastical, Slack, and thousands more — publish their own actions into Shortcuts via App Intents. If an app matters to your workflow, search its name in the action list; the next chapter shows how deep that rabbit hole goes.

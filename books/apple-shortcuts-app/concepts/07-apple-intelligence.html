Chapter 07 · Level Three

The AI Layer: Use Model & Friends

With the OS 26 releases (iOS, iPadOS, and macOS 26), Shortcuts gained something no automation tool on the platform ever had: a language model as an ordinary action. Text goes in, intelligence comes out, and the result flows to the next step like any other variable. The block-builder now has a brain block.

Use Model: three brains, one action

The marquee action is Use Model. You write a request (a prompt), hand it whatever variables you like — a transcript, a web page, a photo's text — and choose which model answers:

ModelWhere it runsBest for
On-Device Your iPhone/iPad/Mac — works offline Quick summaries, reformatting, classification; zero data leaves the device
Private Cloud Compute Apple silicon servers; data never stored, never used for training Bigger reasoning jobs that outgrow the local model, with Apple's privacy guarantees
ChatGPT (extension) OpenAI, via the system integration World knowledge and creative heavy lifting; requires enabling the ChatGPT extension
Note

Ask for follow-up can be enabled on the action to have a short back-and-forth before the answer flows on, and Use Model can be asked to return structured output (a list, a dictionary) so later actions can pick fields out of the response instead of parsing prose.

The Writing Tools actions

Alongside Use Model, the system's Writing Tools became discrete actions you can drop anywhere text flows: Summarize Text, Rewrite Text, Proofread Text, Adjust Tone of Text, Make List from Text, and Make Table from Text. They're narrower than Use Model but faster to configure — no prompt writing, just a transform.

Two more AI actions round out the set: Create Image (Image Playground) generates artwork mid-shortcut, and Open Visual Intelligence jumps into the camera-based lookup experience.

Apple's pre-built intelligent shortcuts

The Gallery's Apple Intelligence section ships ready-made recipes that double as teaching examples — add them, then open them up:

  • Morning Summary — weather, calendar, and reminders distilled into one spoken/notified brief.
  • Summarize PDF / Document Review — hand it any file from the share sheet, get the gist back.
  • Leftover Recipes — photograph the fridge, get dinner ideas. Silly, and a perfect demo of image → model → answer.

Requirements

Apple Intelligence actions need a supported device: iPhone 15 Pro or later (and all iPhone 16/17-class models), iPads and Macs with M1 or newer, or iPad mini with A17 Pro — with Apple Intelligence enabled in Settings. The ChatGPT route additionally requires turning on the ChatGPT extension. On older hardware the rest of this book works fine; this chapter's actions simply don't appear.

Key Idea

The pattern to internalize: deterministic steps gather, the model judges, deterministic steps deliver. Let Get-actions fetch exact data, let Use Model do the one fuzzy step (summarize, decide, rewrite), and let Do-actions handle the consequences. That division keeps shortcuts fast, cheap, and predictable.

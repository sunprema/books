Chapter Three

Boolean Modeling

This is the heart of the whole tool. Take two solids and combine them three ways: glue them together, carve one out of the other, or keep only the overlap. Programmers call this CSG — Constructive Solid Geometry — and it is how nearly every ManifoldCAD model is built.

import {Manifold} from 'manifold-3d/manifoldCAD';
const {cube, cylinder} = Manifold;

const block = cube([40, 40, 20], true);
const hole  = cylinder(20, 8, -1, 64, true);

// Carve the cylinder out of the block:
export default block.subtract(hole);

That's a rounded hole drilled clean through a block — the "hello world" of CAD. Three lines of shapes, one boolean.

Three operations, two ways to write each

Every boolean comes as an instance method (nice for chaining off one shape) and a static function (nice for combining a whole list). They do the same thing.

OperationMethodStaticResult
Uniona.add(b)Manifold.union(a, b)everything in either shape
Differencea.subtract(b)Manifold.difference(a, b)a with b removed
Intersectiona.intersect(b)Manifold.intersection(a, b)only where they overlap

Combining a whole list at once

The static forms accept an array, which is the clean way to merge or carve many shapes — far better than a chain of .add().add().add().

const pillars = [];
for (let i = 0; i < 4; i++) {
  const x = (i % 2) * 30, y = Math.floor(i / 2) * 30;
  pillars.push(cylinder(40, 4).translate([x, y, 0]));
}
const legs = Manifold.union(pillars);          // one solid from many

// Difference with a list subtracts them all from the first:
const plate = Manifold.difference([slab, boltHole1, boltHole2, boltHole3]);
Coplanar faces just work

A classic CAD headache is subtracting a shape whose face lands exactly on another face — the math gets ambiguous and lesser tools produce cracks. Manifold uses symbolic perturbation to resolve these ties deterministically, so coplanar cuts (the Menger-sponge example is built entirely from them) come out clean.

A two-for-one: split

Sometimes you want both the carved result and the piece you carved away — a part and its mating cavity, say. split returns both halves in one pass, cheaper than doing a subtract and an intersect separately.

// intersection first, then difference — in a single operation
const [inside, outside] = result.split(cube(100, true));
Key idea

Design by addition and subtraction. A bracket is a block minus some holes minus a slot. A knob is a cylinder plus a grip texture minus a shaft. Once you see parts as booleans of primitives, modelling becomes arithmetic.

Booleans decide which material exists. Next: where it sits — moving, spinning, and scaling your shapes into place.

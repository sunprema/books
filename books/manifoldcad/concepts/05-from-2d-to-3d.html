Chapter Five

From 2D to 3D

Some shapes are far easier to describe as a flat outline that you then push into the third dimension. Draw a 2D cross-section, then extrude it (push it straight up) or revolve it (spin it around an axis). This is how you make gears, bottles, threads, and letters.

The CrossSection class

A CrossSection is the 2D cousin of Manifold: a set of closed polygons with no self-intersections, guaranteed valid — so 2D booleans are just as robust as 3D ones. It has its own primitives and transforms.

import {CrossSection, Manifold} from 'manifold-3d/manifoldCAD';
const {square, circle} = CrossSection;

const plate = square([40, 40], true)     // 40x40, centred
  .subtract(circle(6).translate([12, 12]))   // punch a hole
  .subtract(circle(6).translate([-12, -12]));
2D constructor / methodMeaning
CrossSection.square(size?, center?)A rectangle; size is a number or [w, h].
CrossSection.circle(radius, segments?)A filled circle.
.offset(delta, joinType?, miterLimit?, segments?)Grow (or shrink, if negative) the outline — how you get rounded corners and even wall thicknesses.
.add / .subtract / .intersectThe same booleans as in 3D, in the plane.

You can also skip the class entirely and hand a raw array of [x, y] points straight to extrude — a "simple polygon". The stretchy-bracelet example builds its profile exactly this way.

Extrude: push a profile upward

// extrude(polygons, height, nDivisions?, twistDegrees?, scaleTop?, center?)
const prism = Manifold.extrude(plate, 10);          // 10mm tall

// A twisted, tapering column — nDivisions gives the twist somewhere to bend:
const drill = Manifold.extrude(square([20, 20], true), 60, 40, 180, 0.3);
//                                          height ^   ^div ^twist ^scaleTop

The extra arguments unlock a lot: nDivisions slices the height so a twist can curve smoothly, twistDegrees spirals the profile as it rises, and scaleTop shrinks (or grows) the top face to make tapers and pyramids.

Revolve: spin a profile around the axis

Place a profile out to one side of the origin and revolve sweeps it around the Y axis — vases, bowls, pulleys, ring gaskets. Give it a partial angle for a wedge.

// revolve(polygons, circularSegments?, revolveDegrees?)
const profile = CrossSection.circle(4).translate([20, 0]);
const torus   = Manifold.revolve(profile, 64);        // full ring
const scoop   = Manifold.revolve(profile, 64, 270);   // 3/4 sweep
Going the other way: 3D → 2D

Two Manifold methods return a CrossSection: slice(z) cuts a flat section at a given height, and project() flattens the whole solid onto the X-Y plane (its shadow). The auger example extrudes bead.slice(0) — it slices a 3D bead and re-extrudes that outline. Handy for generating a matching base plate or a 2D drawing.

Key idea

If a shape has a natural profile — a constant cross-section, or a silhouette spun around an axis — reach for 2D first. Designing in the plane and lifting into 3D is simpler and more robust than wrestling the full solid.

Chapter Seven

Smooth & Organic Shapes

Cubes and cylinders get you mechanical parts. For hearts, knots, ripples, and lattices you need to bend the mesh itself. Three tools do this: warp moves every vertex by a function, refine adds vertices so there's something to bend, and smooth rounds a coarse mesh into flowing curves.

warp: a function applied to every vertex

warp hands your callback each vertex position and lets you mutate it in place. No triangles, no bookkeeping — just math on points. Because the topology doesn't change, the result stays a valid solid.

// Push a flat sheet into a gentle wave along X.
const sheet = Manifold.cube([80, 80, 2], true).refine(80);

const wavy = sheet.warp((v) => {
  v[2] += 6 * Math.sin(v[0] / 10);   // v is [x, y, z]; edit it directly
});

export default wavy;
Refine before you warp

A warp only moves the vertices that exist. A plain cube has 8 of them, so bending it does nothing visible. refine(n) splits every edge into n pieces first, giving the surface enough points to curve smoothly. More refinement = smoother warp = bigger mesh.

This is exactly how the built-in Heart example works: it starts from a high-segment sphere and warps each point outward by solving Taubin's heart equation with Newton's method. And the Torus Knot revolves a circle, then warps the resulting tube into a knot — no triangle ever touched by hand.

smooth: round off a coarse mesh

Where warp is explicit math, Manifold.smooth interpolates a low-poly mesh into a curved one, and lets you keep chosen edges sharp. Pair it with refine to actually generate the rounded triangles.

// smooth(mesh, sharpenedEdges?) then refine to realise the curve
const blob = Manifold.smooth(mesh, sharpenedEdges).refine(n);

The Scallop example builds a fan of triangles by hand, marks each rib's smoothness, then smooth().refine(50) turns that skeleton into a rippled shell. There's also smoothOut(minSharpAngle, minSmoothness) to round a mesh using only its own geometry, and smoothByNormals when you have normal data.

Colour by curvature (a taste of properties)

setProperties attaches per-vertex data — most commonly vertex colours. Combined with calculateCurvature, you can paint a surface by how sharply it bends:

const shell = Manifold.smooth(mesh, edges)
  .refine(50)
  .calculateCurvature(-1, 0)                 // write mean curvature to property 0
  .setProperties(3, (color, pos, oldProp) => {
    const t = Math.max(0, Math.min(1, oldProp[0] / 3 + 0.5));
    color[0] = t; color[1] = 0; color[2] = 1 - t;   // blue → red
  });
Beyond meshes: level sets

For truly implicit shapes — gyroids, metaballs, blends — Manifold.levelSet(sdf, bounds, edgeLength) meshes the zero-crossing of a signed distance function you write in JavaScript. It's how the Gyroid Module example builds an infinite minimal surface from one short formula.

Key idea

Organic modelling in ManifoldCAD is procedural: describe the deformation as a function of position, give the surface enough vertices to express it, and let the engine keep the result watertight. Sculpting becomes programming.

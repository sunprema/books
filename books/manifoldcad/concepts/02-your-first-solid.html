Chapter Two

Your First Solid

Let's write a complete, runnable ManifoldCAD script — all four lines of it — then take apart every piece. Paste this into the editor at manifoldcad.org and you'll see a box on screen.

import {Manifold} from 'manifold-3d/manifoldCAD';

const box = Manifold.cube([30, 20, 10], true);

export default box;

Three ideas are doing all the work here, and you'll use them in every script you ever write:

  1. Import the Manifold class from the built-in module.
  2. Call a constructor (Manifold.cube) to make a solid.
  3. Export that solid as the default — that's what gets rendered.
Tip: destructure the constructors

Typing Manifold. over and over gets old. Most examples pull the ones they need into locals: const {cube, sphere, cylinder} = Manifold; — then you just call cube(...).

The four primitives

Every model starts from these building blocks. Note the exact signatures — the defaults matter.

ConstructorMakes
cube(size?, center?)A box. size is a number (a cube) or [x, y, z]. Default sits in the first octant (a corner at the origin); pass center=true to centre it.
sphere(radius, segments?)A sphere centred at the origin. segments sets the facet count (its smoothness).
cylinder(height, radiusLow, radiusHigh?, segments?, center?)A cylinder — or a cone if the two radii differ. A handy trick: pass radiusHigh = -1 to mean "same as radiusLow".
tetrahedron()A unit tetrahedron — the simplest closed solid, useful for tests.
 
const {cube, sphere, cylinder} = Manifold;

const block = cube(20);                 // 20mm cube, corner at origin
const ball  = sphere(12, 64);           // radius 12, 64 segments -> smooth
const rod   = cylinder(30, 5);          // height 30, radius 5
const cone  = cylinder(30, 10, 0);      // radiusHigh 0 -> a point -> a cone
const funnel = cylinder(20, 6, 12);     // radiusLow 6, radiusHigh 12 -> a frustum

Resolution: how round is round?

A sphere on a computer is really a many-sided polyhedron. The segments argument controls how many facets it has — more segments, smoother surface, bigger mesh. When you leave it out, Manifold picks a sensible default based on the shape's size, which you can tune globally:

// Set the global level-of-detail once, at the top of your script.
setMinCircularAngle(3);         // max 3 degrees between facets
setMinCircularEdgeLength(0.5);  // no facet edge shorter than 0.5mm
// or force an exact count for every curved primitive:
setCircularSegments(64);

const smooth = sphere(20);      // now honours the settings above
Key idea

Set resolution globally at the top, not per-shape. Then a single knob controls the trade-off between a fast, chunky preview and a slow, glass-smooth final export — across your whole model at once.

Don't forget the export

A script with no export default renders nothing — the most common "why is my screen blank?" mistake. Whatever you want to see, it has to be the thing you export.

You can make individual solids now. Next we'll do the thing that turns a pile of primitives into a real part: combining them.

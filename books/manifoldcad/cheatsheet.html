Quick Reference

The 30-Second ManifoldCAD

Everything you reach for most, on one open spread. Units are millimetres; Z is up; rotations are in degrees; every method returns a new solid.

Script skeleton

import {Manifold, CrossSection, GLTFNode,
        show, only} from 'manifold-3d/manifoldCAD';
const {cube, sphere, cylinder} = Manifold;

// build...
export default result;   // Manifold | GLTFNode | array | function

Primitives (3D)

cube([x,y,z], center?)          // box; center=true to centre
sphere(radius, segments?)
cylinder(h, rLow, rHigh?, seg?, center?)  // rHigh=-1 -> = rLow; 0 -> cone
tetrahedron()
Manifold.hull([p0, p1, ...])    // convex hull of points/manifolds

Booleans (CSG)

a.add(b)        Manifold.union([a,b,c])
a.subtract(b)   Manifold.difference([a,b,c])   // b,c cut from a
a.intersect(b)  Manifold.intersection([a,b])
const [in_, out] = a.split(cutter);   // subtract + intersect at once
a.trimByPlane([nx,ny,nz], offset);    // keep one side of a plane

Transforms (chainable)

.translate([x,y,z])     .rotate([x,y,z])   // DEGREES, X→Y→Z
.scale(s | [x,y,z])     .mirror([nx,ny,nz])
.transform(mat4)        .warp(v => { v[0]+=…; })  // edit vertex in place

2D → 3D

CrossSection.square([w,h], center?)   CrossSection.circle(r, seg?)
cs.offset(d)   cs.add/subtract/intersect(other)
Manifold.extrude(poly, h, nDiv?, twistDeg?, scaleTop?, center?)
Manifold.revolve(poly, seg?, revolveDeg?)
solid.slice(z)   solid.project()      // 3D -> CrossSection

Smooth & organic

.refine(n)                 // subdivide edges BEFORE warping
.warp(v => { … })          // move every vertex by a function
Manifold.smooth(mesh, sharpenedEdges).refine(n)
.smoothOut(minSharpAngle?, minSmoothness?)
Manifold.levelSet(sdf, bounds, edgeLength)   // mesh an SDF

Resolution (set once, at top)

setMinCircularAngle(3);        // degrees between facets
setMinCircularEdgeLength(0.5); // min facet edge, mm
setCircularSegments(64);       // or force an exact count

Scene, colour, animation

const n = new GLTFNode(parent?);
n.manifold = solid;
n.material = { baseColorFactor:[r,g,b], metallic:0, roughness:1,
               attributes:['COLOR_0'] };   // COLOR_0 = per-vertex colour
n.translation = [x,y,z];  n.rotation = […];  n.scale = […];
setMorphStart(m, fn);  setAnimationDuration(5);  setAnimationMode('ping-pong');
export default getGLTFNodes();   // or a node / array of nodes

Ask the model

.volume()   .surfaceArea()   .numTri()   .genus()
.boundingBox()   // { min:[x,y,z], max:[x,y,z] }
.isEmpty()       console.log(...)  // output panel, lower-right

Debug (like OpenSCAD # and *)

show(shape)   // this + copies in transparent RED
only(shape)   // show only this, ghost everything else grey
Top gotchas

① No export default → blank screen. ② Warp does nothing without a refine() first. ③ Rotations are degrees, not radians. ④ Give thin walls real thickness — marginal geometry wobbles under floating point. ⑤ Use GLTFNode, not union, for parts that don't touch.

The whole tool in one line

Make valid solids from primitives & profiles, add/subtract/intersect them, transform copies into place, warp for organic form, and export a watertight .glb/.3mf — all offline, all in code.

Grounded in the ManifoldCAD User Guide (manifoldcad.org/docs/jsuser) and the editor's shipped example scripts. Verify signatures live by typing Manifold. in the editor.

09 The Outside World

Reaching the Web

Two tools reach past your machine. WebSearch finds pages; WebFetch reads one. They're a pair — search returns links, then fetch follows them. Both ask first.

WebSearch: links, not pages

WebSearch runs a query against Anthropic's web-search backend and returns result titles and URLs — it does not fetch the pages. A single call may issue up to eight backend searches, refining internally before returning. Scope it with allowed_domains or blocked_domains (never both in one call).

find, then read❯ WebSearch  query="ripgrep multiline flag"
  # → titles + URLs only
❯ WebFetch   url="https://…"  prompt="how to enable -U"
▣
Availability & rules

The backend isn't configurable — for a different provider, add an MCP search tool (chapter 11). Permission rules take no specifier: a bare WebSearch in allow/deny is the only form. Available on the Claude API and Microsoft Foundry, on Vertex AI with Claude 4 models; not on Amazon Bedrock.

WebFetch: lossy by design

WebFetch takes a URL and a prompt. It fetches the page, converts HTML to Markdown, and runs your prompt against the content using a small, fast model. Crucially, Claude usually receives that model's answer, not the raw page.

⚠
"The page doesn't mention X"

That might only mean your prompt didn't ask about X. The extraction step decides what reaches Claude, so a negative result isn't proof. Re-fetch with a sharper prompt — or use curl via Bash for the unprocessed page.

A few behaviors shape what comes back:

  • HTTP URLs are upgraded to HTTPS automatically
  • Large pages are truncated before processing
  • Responses are cached for 15 minutes
  • A cross-host redirect isn't followed — WebFetch returns the target URL and Claude fetches it again
★
Domains & prompts

In default mode WebFetch prompts the first time it hits a new domain, except a built-in set of preapproved docs domains. Pre-allow others with WebFetch(domain:example.com); an explicit deny/ask/allow rule overrides the preapproved set. auto and bypassPermissions modes skip the prompt entirely.

Rule format

ToolName(specifier)
Bash(npm run *)   # +Monitor
Read(~/x/**)     # +Grep,Glob,LSP
Edit(/src/**)   # +Write,Notebook
WebFetch(domain:x.com)

Edit(...) also grants read. Hooks use bare names.

Top gotchas

  • Grep skips .gitignore; Glob doesn't
  • ripgrep regex — escape {} etc.
  • Edit needs an exact, unique match
  • Write overwrites the whole file
  • WebFetch returns an extract, not the page
  • Bash export doesn't persist

Hosted-only

Not on Bedrock / Vertex / Foundry: Monitor, PushNotification, SendUserFile, RemoteTrigger, ScheduleWakeup. WebSearch is missing on Bedrock too.

The 30-second model

Claude only acts through named tools. Each call is matched against your allow/deny rules; deny wins, allow skips the prompt, else the tool's default decides. Grow the set with MCP.
